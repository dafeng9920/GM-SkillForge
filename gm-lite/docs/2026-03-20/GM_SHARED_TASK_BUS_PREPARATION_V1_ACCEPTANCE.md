# GM Shared Task Bus Preparation v1 Acceptance

## 通过标准
1. `.gm_bus` 目录结构职责定义清晰
2. 六个核心协议对象边界清晰
3. `manifest / task board / projection / authority` 关系清晰
4. 与 `GM-SkillForge`、`D:/NEW-GM` 的边界清晰
5. Light 版禁止项清晰
6. 三权任务卡完整
7. 所有回收件按标准路径归档
8. 无阻断性越界问题

## 退回标准
1. 直接进入运行时实现
2. 直接进入插件直连
3. 把 task board 定义成权威状态写源
4. 直接复制旧仓库实现
5. 任务卡或回填路径不完整

## 局部修正后再审
- 命名不一致
- 文档小节缺失
- 任务依赖标注不够清楚
- 写回路径表达不完整
