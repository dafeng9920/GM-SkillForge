# GM LITE Controller Console Minimal Implementation v1 Acceptance

## 通过标准
1. 控制台最小目录骨架已落位
2. 读取对象与视图模型骨架已存在
3. 状态 / 告警 / gate-ready 视图骨架已存在
4. 主控动作占位与最小使用说明已存在
5. 未混入 UI / watcher / adapter / runtime
6. 四个子任务三件套齐全

## 退回标准
1. 混入 UI 实现
2. 混入 watcher / adapter / runtime
3. 控制台被写成权威状态源
4. 交付物与准备边界明显不一致
5. 回收件不完整
