# Development Scripts Directory

This directory contains scripts for development and testing.

## Scripts

### API Development
- `simple_api.py` - Simple FastAPI development server
- `run_api.py` - API server launcher
- `start_full_api.py` - Full-featured API server starter
- `start_backend.py` - Backend service launcher

## Usage

```bash
# Start development server
python scripts/dev/simple_api.py

# Start full API server
python scripts/dev/start_full_api.py
```

## Environment

These scripts are intended for development only and should not be used in production.
