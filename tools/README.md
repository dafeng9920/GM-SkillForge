# Tools Directory

This directory contains utility scripts for development and maintenance.

## Scripts

### Template Utilities
- `chat_to_md.py` - Convert chat logs to markdown format
- `extract_template.py` - Extract template data from files
- `extract_template_v2.py` - Extract template data (v2)
- `heal_template.py` - Repair and validate template files

## Usage

Each script can be run directly:
```bash
python tools/chat_to_md.py <input_file>
```
