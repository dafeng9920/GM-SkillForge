# 🦞 OpenClaw 私人 AI 助理使用手册 (老板简版)

欢迎使用你的专属 OpenClaw 助理！这不仅是一个聊天机器人，它是运行在你私有环境下的、具备执行能力的“数字员工”。

## 一、 如何召唤它？
在 Discord 授权的频道中：
*   **直接对话**：像和真人聊天一样输入指令。
*   **@它**：如果你想在群聊中单独叫它。

## 二、 它能帮你做什么？
1.  **深度研究**：
    *   “帮我分析一下小红书最近大火的 AI 助理赛道，给我出个报告。”
    *   “找一下这几个竞品的最新动态。”
2.  **代码与网页**：
    *   “帮我写个简单的落地页代码。”
    *   “帮我写个自动化表格处理脚本。”
3.  **任务执行**：
    *   它具备控制本地浏览器和沙箱运行代码的能力，可以帮你执行一些重复性的繁琐工作。

## 三、 安全承诺 (你的数据不出门)
*   **私有部署**：所有对话和处理的数据都存储在你自己的服务器/电脑上。
*   **高标准加固**：我们已为你锁死了外部攻击通道，确保只有授权的你和你的频道可以调动它。

## 四、 温馨提示
1.  **不要在公共场合泄露 Token**：那是它的“身份钥匙”。
2.  **持续进化**：你可以通过“技能系统”不断给它增加新的超能力。
