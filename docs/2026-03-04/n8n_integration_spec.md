# n8n Integration Spec: Cross-App Automation via OpenClaw

## Overview
This specification details how OpenClaw utilizes the `n8n-workflow-automation` skill to execute complex, cross-application workflows.

## Workflow Patterns

### 1. Email to Notion/Trello (Example)
- **Trigger**: OpenClaw detects high-priority email.
- **Action**: Generate n8n JSON via `n8n-workflow-automation` skill.
- **Node 1**: `Notion Node` - Save email body to "Inbox" database.
- **Node 2**: `Zapier Node` - Trigger external notification flow.
- **Node 3**: `Trello Node` - Create card in "To Do" list.

## Integration Interface

### CLI Command (via ClawHub)
```bash
npx clawhub@latest install n8n-workflow-automation
```

### Configuration
- **Webhook URL**: To be provided in `openclaw.json` or `.env`.
- **API Key**: Required for n8n core authentication.

## Security & Auditing
- **Auditable JSON**: Every workflow generated by the skill includes idempotent IDs.
- **Human-in-the-Loop**: Large or destructive workflows require manual review in the n8n queue.
