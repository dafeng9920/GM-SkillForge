## 策略核心概念

**三因子模型**:

| 因子 | 说明 | 常用指标 |
|------|------|----------|
| **Momentum** | 价格趋势动量 | Price Return, ROC |
| **Value** | 估值便宜程度 | P/E, P/B, EV/EBITDA |
| **Quality** | 财务质量 | ROE, Debt/Equity, Margin |

**Composite Score**:
```
CompositeScore = w1 * Momentum + w2 * Value + w3 * Quality
```

**Portfolio Weights**:
- 根据 CompositeScore 分配权重
- 高分股票获得更高权重
