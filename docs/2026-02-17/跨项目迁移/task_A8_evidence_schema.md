## Evidence Schema 核心概念

**StandardEvidence 类型**:

| 类型 | 说明 | 必要字段 |
|------|------|----------|
| `ScanReport` | 扫描报告 | component_id, status, findings, timestamp |
| `TestResult` | 测试结果 | test_id, passed, duration, error_message |
| `Tombstone` | 废弃记录 | skill_id, revoked_at, reason, revoked_by |

**Schema 设计原则**:
1. 所有字段必须有明确类型
2. 必填字段 vs 可选字段
3. 默认值定义
4. 验证规则（正则、范围、枚举）
