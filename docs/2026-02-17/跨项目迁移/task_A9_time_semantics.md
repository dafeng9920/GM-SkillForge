## 时间语义核心概念

**三种时间**:

| 时间类型 | 说明 | 示例 |
|----------|------|------|
| `EventTime` | 事件实际发生时间 | 财报发布日期 2024-01-15 |
| `AvailableTime` | 数据可用时间 | 数据入库时间 2024-01-20 |
| `DecisionTime` | 决策时间 | 交易决策时间 2024-01-21 |

**核心约束**:
```
EventTime <= AvailableTime <= DecisionTime
```

**违反约束的后果**:
- **Look-Ahead Bias**: 使用了未来数据
- **Backtest 失真**: 回测结果不可靠
- **生产环境崩溃**: 实时数据不可用

## 使用场景

| 场景 | 说明 |
