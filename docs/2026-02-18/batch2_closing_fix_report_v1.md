### 2.2 决策语义修订（问题 B）

**Before**:
```
final_gate_decision: PASSED
permit_token: null (dry-run)
release_allowed: false
```
问题：PASSED + permit null 语义不清，容易误判为"通过可发布"

**After**:
```
final_gate_decision: PASSED_NO_PERMIT
permit_token: null (dry-run)
release_allowed: false
release_blocked_by: PERMIT_REQUIRED
fail_closed_enforced: true
```

**判定来源**:
- `release_allowed=false` 来源：Fail-Closed 规则 `NO_PERMIT_NO_RELEASE`
- 无 permit 不可发布属于**硬约束**（fail-closed）
- 即使所有 Gate 返回 ALLOW，缺少 permit 仍阻止发布
