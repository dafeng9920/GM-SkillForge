# 最终落地完成前置清单

- 日期：`2026-03-10`
- 适用对象：`GM-SkillForge 从样本板走向最终落地完成`
- 作用：把“最小最终落地三步版”展开为可执行检查清单

## 一、目标定义

本清单的唯一目标是确认：  
GM-SkillForge 是否已经具备宣告“最终落地完成”的最低必要条件。

在本清单全部通过前，统一只能使用以下口径：

- `样本已封板`
- `patch_proposal_only`
- `not absorbed`
- `not final accepted`

不得提前写成：

- `已最终落地完成`
- `主仓已完全吸收`
- `final accepted batch`

## 二、第一步前置清单：唯一主链是否写死

### 必须明确的唯一主链

- [ ] 已明确唯一标准主链为：
  - `Permit -> dropzone -> pre_absorb_check -> absorb -> local accept -> final accept`

### 必须明确的禁止规则

- [ ] 已明确：没有 `Permit` 不得推进
- [ ] 已明确：未通过 `pre_absorb_check` 不得 `absorb`
- [ ] 已明确：未 `absorb` 不得写 `final accepted`
- [ ] 已明确：`proposal-only` 不得冒充 `absorbed`
- [ ] 已明确：`ABSORB_READY (proposal only)` 不得冒充 `final accepted`

### 必须明确的状态词

- [ ] 已统一 `proposal-only`
- [ ] 已统一 `ABSORB_READY (proposal only)`
- [ ] 已统一 `absorbed`
- [ ] 已统一 `final accepted`

## 三、第二步前置清单：是否存在一单真实落地主单

### 任务资格

- [ ] 已选定一单正式落地主单
- [ ] 该单不是失败样本
- [ ] 该单不是 remediation 样本
- [ ] 该单不是仅 `proposal-only` 样本

### 执行链完整性

- [ ] 该单已有 `Permit`
- [ ] 该单已进入 `dropzone`
- [ ] 该单已通过 `pre_absorb_check`
- [ ] 该单已完成真实 `absorb`
- [ ] 该单已完成 `local accept`
- [ ] 该单已完成 `final accept`

### 证据链完整性

- [ ] 该单已有清晰任务标识
- [ ] 该单已有完整门禁记录
- [ ] 该单已有 absorb 证据
- [ ] 该单已有 accept 证据
- [ ] 该单已有最终结论可回放材料

## 四、第三步前置清单：正式结论文档是否成立

### 文档最少必须回答的问题

- [ ] 已明确哪一单完成真实 `absorb`
- [ ] 已明确哪一天完成 `final accept`
- [ ] 已明确为什么从该时点起可称“最终落地完成”

### 文档口径要求

- [ ] 未混入 `proposal-only` 口径
- [ ] 未把样本冻结误写为最终落地
- [ ] 未把 `ABSORB_READY (proposal only)` 误写为 `final accepted`
- [ ] 已说明主仓 authoritative 更新已经发生

## 五、系统补强前置项

这些项理论上可以与三步并行，但若缺失，会显著削弱“最终落地完成”的可信度。

- [ ] `Permit` 已成为唯一不可绕过入口
- [ ] 三哈希已并入主链：
  - `demand_hash`
  - `contract_hash`
  - `decision_hash`
- [ ] 最小运行接口已统一：
  - `RunRequest`
  - `RunResult`
  - `ArtifactManifest`
- [ ] 云端执行环境口径已统一
- [ ] 运输层不再完全依赖手工中继

## 六、通过判定

只有当以下三组条件同时满足，才可宣布“最终落地完成”：

- [ ] 第一组：唯一主链已写死
- [ ] 第二组：一单真实落地主单已走完
- [ ] 第三组：正式结论文档已成立

## 七、当前状态备注

截至 `2026-03-10`，当前更接近：

- `样本证明已完成`
- `修复样本已出现`
- `最终主链尚未正式走完`

因此当前仍不能宣告：

- `最终落地完成`

## 八、一句话

这份清单的作用，不是增加复杂度，  
而是避免在还没真实 `absorb + final accept` 之前，提前宣布胜利。
