## 1. Governance Candidate Intake

- Minimum implementation form:
  - typed payload object
- Interface treatment:
  - remain interface placeholder only
- Not allowed in minimal implementation:
  - governance decision semantics
  - execution/service behavior

## 2. Governance Validation Intake

- Minimum implementation form:
  - typed payload object
- Interface treatment:
  - remain interface placeholder only
- Not allowed in minimal implementation:
  - governance validation result semantics
  - review/release semantics

## 3. Pre-Packaging Review Intake

- Minimum implementation form:
  - typed payload object
- Interface treatment:
  - remain interface placeholder only
- Not allowed in minimal implementation:
  - release permit semantics
  - packaging execution semantics

## 4. Shared bridge boundary

- Payloads may enter minimal implementation.
- Interfaces stay in placeholder state.
- No intake service, no handler, no workflow binding.
