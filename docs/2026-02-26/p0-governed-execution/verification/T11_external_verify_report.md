## 3. 验真脚本功能

### 3.1 验证步骤

```
1. Schema Validation    - 验证 envelope 结构完整性
2. Signature Verification - 验证 Ed25519 签名真实性
3. Timestamp Validation - 验证时间戳有效性（未过期）
4. Nonce Validation     - 验证 nonce challenge（如有）
5. Body Hash Verification - 验证 body 内容完整性
```

### 3.2 使用方式

```bash
# 使用 hex 公钥验证
python verify_evidence.py \
    --package sample_evidence.json \
    --pubkey abc123def456...

# 使用 PEM 文件验证
python verify_evidence.py \
    --package sample_evidence.json \
    --pubkey node_public.pem

# 详细输出
python verify_evidence.py \
    --package sample_evidence.json \
    --pubkey node_public.pem \
    --verbose

# 静默模式（用于脚本）
python verify_evidence.py \
    --package sample_evidence.json \
    --pubkey node_public.pem \
    --quiet
```

### 3.3 独立性

脚本 **不依赖** SkillForge 内部模块，仅需：
- Python 3.8+
- cryptography 库
- 标准库

第三方可以直接使用此脚本，无需安装整个 SkillForge 项目。
