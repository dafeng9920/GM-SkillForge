### 交付物摘要

1. **Schema 校验器 (SchemaValidator)**
   - 验证 NormalizedSkillSpec 符合 JSON Schema
   - 支持类型检查、模式匹配、枚举值验证
   - 支持长度和数值范围验证

2. **Contract 校验器 (ContractValidator)**
   - 验证 input_contract 和 output_contract 结构
   - 检查 schema_type 有效性
   - 验证 required 字段都在 properties 中定义

3. **一致性校验器 (ConsistencyValidator)**
   - 验证版本格式 (semver)
   - 验证 skill_id 格式
   - 验证 spec_hash 格式 (SHA-256)
   - 验证依赖结构一致性

4. **错误码体系 (E3xx 系列)**
   - E301-E310: Schema 验证错误
   - E311-E316: Contract 验证错误
   - E321-E327: 一致性验证错误

5. **ValidationReport 输出**
   - is_valid: 验证是否通过
   - failures: 错误列表（含 code, message, field_path, suggested_fix）
   - warnings: 警告列表
   - validated_at, validator_version, failure_count, warning_count
