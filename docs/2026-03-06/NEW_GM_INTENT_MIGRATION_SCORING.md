## 评分模型

三维定义（0-100）：

- 契合度（Fit）：与当前 `contracts-first + 8-gate + evidence` 架构的匹配程度
- 贡献度（Contribution）：对当前主线交付与治理闭环的直接贡献
- 价值度（Value）：中长期复用价值、战略价值、复利价值

综合分公式：

`Score = 0.35 * Fit + 0.35 * Contribution + 0.30 * Value`

阈值：

- `>= 80`：优先迁移
- `65-79`：选择性迁移
- `< 65`：暂不迁移（后置）

## 结论：是否有必要“全量迁移”？

**结论：没有必要。**

分布：

- 优先迁移（`>=80`）：8 项
- 选择性迁移（`65-79`）：6 项
- 暂不迁移（`<65`）：4 项

贡献度口径下（18 项总贡献度）：

- 优先 + 选择性集合约覆盖 `82%` 贡献度
- 暂不迁移集合约 `18%`

所以最优策略是：**高分必迁 + 中分择优 + 低分后置**，而不是全量迁移。
