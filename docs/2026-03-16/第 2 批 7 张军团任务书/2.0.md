## 统一交付要求

每个任务书执行完成后，执行者必须在对应任务文档中回填：

1. `ExecutionReport`
2. `EvidenceRef`
3. `实测脚本/命令`
4. `交付物路径`
5. `Gate 自检结果`
6. `阻塞项`（如有）

同时必须回填本总控板 `2.0.md` 中对应任务的三权记录。
要求：
- `executor` 只回填 `Execution`
- `reviewer` 只回填 `Review`
- `compliance_officer` 只回填 `Compliance`
- 三者不得互相代填
- 若未按角色回填或格式不规范，主控官直接判 `REQUIRES_CHANGES`
