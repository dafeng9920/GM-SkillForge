## 证据等级 (E1-E5)

| 等级 | 名称 | 强度 | 可重现 | Gate | 最小证据 |
|------|------|------|--------|------|----------|
| E1 | Declaration | 1 | 否 | 否 | 仅声明/注释 |
| E2 | Log Output | 2 | 否 | 否 | 日志文件 |
| E3 | Structured Artifact | 3 | 否 | 否 | 结构化文件 |
| E4 | Replayable Test | 4 | 是 | 否 | 可执行测试 |
| E5 | Gate Decision | 5 | 是 | 是 | dual-gate 挂载 |

## 合规原则

1. **无默认全覆盖**: 不在 `covered_items` 中声明的项 = 未覆盖
2. **无默认证据强度**: 必须显式声明 E1-E5 等级
3. **证据引用必需**: 已覆盖项必须提供至少一个 evidence_ref
4. **排除不计入**: `exclusions` 中的项不计入覆盖率统计
