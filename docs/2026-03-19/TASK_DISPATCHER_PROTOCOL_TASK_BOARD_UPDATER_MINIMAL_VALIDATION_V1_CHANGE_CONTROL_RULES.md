# Task Dispatcher Protocol Task Board Updater Minimal Validation v1 Change Control Rules

## 本阶段允许记录的变更类型
- 状态推导结论记录
- anomaly 识别结论记录
- 非阻断性格式问题记录

## 本阶段禁止直接执行的变更类型
- 不直接扩展更多任务板格式
- 不直接加入自动发送
- 不直接加入最终裁决逻辑

## 非阻断性问题后续处理边界
- markdown 表格格式规范化，可放到后续增强回合
- 更多任务板格式兼容，可放到后续增强回合

## 下一阶段前不得触碰的实现面
- 自动发送
- 自动终验
- 数据库存储
- 外部系统接入
