# 三位一体：终极智慧

> **核心理念**: 摸着机构过河，看着龙头下单，跟着大佬摸鱼。

## 一、三位一体的智慧

| 智慧 | 含义 | 解决问题 |
|:---|:---|:---|
| **摸着机构过河** | 机构是石头，踩着他们的脚印走 | 安全性 |
| **看着龙头下单** | 龙头是方向，只做最强的 | 确定性 |
| **跟着大佬摸鱼** | 大佬出力，我们跟着获利 | 效率性 |

## 三、终极操作流程

```
🎯 三位一体 · 每日操作流程 🎯

开盘前（5分钟）：
  1. 看龙虎榜 → 哪些机构在买？
  2. 看板块涨幅 → 谁是龙头？
  3. 看大佬持仓 → 他们在买什么？
  → 得到候选池

盘中（实时）：
  1. 候选池中谁在领涨？ → 确认龙头
  2. 机构是否持续流入？ → 确认安全
  3. 大佬是否加仓？ → 确认方向
  → 三者共振 → 买入

收盘后（10分钟）：
  1. 机构还在买吗？ → 决定持有
  2. 还是龙头吗？ → 决定去留
  3. 大佬动了吗？ → 决定操作
  → 更新策略
```
