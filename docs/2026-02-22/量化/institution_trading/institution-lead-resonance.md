# 机构与龙头共振：交易圣杯

> **核心理念**: 机构是钱，龙头是势。钱和势共振，就是必涨。

## 一、什么是共振？

| 情况 | 结果 | 概率 |
|:---|:---|:---|
| 机构买 + 不是龙头 | 可能涨，但慢 | 60% |
| 是龙头 + 机构不买 | 可能涨，但虚 | 50% |
| **机构买 + 是龙头** | **必涨** | **90%+** |

**核心逻辑**：
- 机构买，说明**基本面、资金面**没问题
- 是龙头，说明**技术面、情绪面**没问题
- 两者共振，所有维度都指向上涨

## 二、共振四等级

| 等级 | 特征 | 操作 |
|:---|:---|:---|
| **S级共振** | 3家以上顶级机构 + 板块第一龙头 + 突破新高 | 🚀 重仓买入，持股待涨 |
| **A级共振** | 2家以上机构 + 涨幅前3 + 突破箱体 | 📈 半仓买入，设好止损 |
| **B级共振** | 1家机构 + 涨幅前10 + 突破均线 | 👀 轻仓参与，严格止损 |
| **无共振** | 不同时满足 | ⏳ 不看、不碰、不关心 |

## 四、三条铁律

```
⚡ 共振交易三条铁律 ⚡

1. 没有机构，不买
   再好的龙头也可能是陷阱

2. 不是龙头，不买
   再多的机构也可能是接盘

3. 不同时满足，不买
   宁愿错过，不要过错
```

## 六、核心价值

| 维度 | 机构解决 | 龙头解决 |
|------|----------|----------|
| **能不能涨** | ✅ 基本面、资金面 | - |
| **涨得快不快** | - | ✅ 技术面、情绪面 |
| **什么时候涨** | - | ✅ 趋势确认 |

**机构解决"能不能涨"的问题，龙头解决"涨得快不快"的问题，共振解决"什么时候涨"的问题。三者合一，就是必涨。**
