### 3.4 验证规则

```python
验证类型:
- 时间验证: 信号持续至少3个tick或60秒
- 空间验证: 突破幅度至少2%，放量至少1.5倍
- 共振验证: 至少3个信号同时指向同一方向
- 诱多排除: 检测假突破、放量滞涨、资金背离
- 验证点机制: 买入后30分钟内要求涨1%，未达成即离场
```

### 7.8 机构与龙头共振：交易圣杯 ⭐⭐⭐

**核心理念**: 机构是钱，龙头是势。钱和势共振，就是必涨。

#### 什么是共振？

| 情况 | 结果 | 概率 |
|:---|:---|:---|
| 机构买 + 不是龙头 | 可能涨，但慢 | 60% |
| 是龙头 + 机构不买 | 可能涨，但虚 | 50% |
| **机构买 + 是龙头** | **必涨** | **90%+** |

**核心逻辑**：
- 机构买，说明**基本面、资金面**没问题
- 是龙头，说明**技术面、情绪面**没问题
- 两者共振，所有维度都指向上涨

#### 共振四等级

| 等级 | 特征 | 操作 |
|:---|:---|:---|
| **S级共振** | 3家以上顶级机构 + 板块第一龙头 + 突破新高 | 🚀 重仓买入，持股待涨 |
| **A级共振** | 2家以上机构 + 涨幅前3 + 突破箱体 | 📈 半仓买入，设好止损 |
| **B级共振** | 1家机构 + 涨幅前10 + 突破均线 | 👀 轻仓参与，严格止损 |
| **无共振** | 不同时满足 | ⏳ 不看、不碰、不关心 |

#### 共振信号检测

| 机构信号 🟢 | 龙头信号 🔴 |
|:---|:---|
| 龙虎榜3家以上机构买入 | 板块涨幅第一 |
| 北向连续10日加仓 | 成交量板块第一 |
| 大宗交易溢价 | 技术形态突破 |
| 机构研报密集 | 市场情绪热点 |
| 机构持仓增加 | 领涨市场核心 |

#### 三条铁律

```
⚡ 共振交易三条铁律 ⚡

1. 没有机构，不买
   再好的龙头也可能是陷阱

2. 不是龙头，不买
   再多的机构也可能是接盘

3. 不同时满足，不买
   宁愿错过，不要过错
```

**核心价值**：机构解决"能不能涨"的问题，龙头解决"涨得快不快"的问题，共振解决"什么时候涨"的问题。三者合一，就是必涨。

### 7.10 红海淘金：机构票筛选系统 ⭐⭐⭐

**核心理念**: 机构不是大海淘沙，是红海淘金。90%的机构票不用看，你只需要盯那10%。

#### 为什么机构票也要筛选？

| 误区 | 真相 |
|:---|:---|
| 所有机构都聪明 | **机构也会错，也会踩雷** |
| 机构买我就买 | **机构买的时候你看不到，看到时可能已在高位** |
| 机构多就是好 | **机构抱团取暖，也可能抱团踩踏** |
| 龙虎榜全是机会 | **龙虎榜也有假机构、一日游、假突破** |

**核心**：不是看"机构买了什么"，而是看**什么样的机构买了什么样的股票**。

#### 机构的分类

| 类型 | 特征 | 风格 | 参考价值 | 跟随策略 |
|:---|:---|:---|:---|:---|
| **顶级机构** | 社保、养老、QFII、顶级公募 | 长期投资，左侧交易 | ⭐⭐⭐⭐⭐ | 他们买的时候是低点，可以等回调跟 |
| **主力机构** | 大型公募、保险、券商自营 | 波段操作，右侧交易 | ⭐⭐⭐⭐ | 他们买的时候是启动点，可以跟随 |
| **游资机构** | 一线游资、私募 | 短线炒作，快进快出 | ⭐⭐⭐ | 他们买的时候是高潮点，要跑得快 |
| **假机构** | 一日游、对倒、假席位 | 骗线、诱多、出货 | ⭐ | 别跟，离远点 |

#### 机构票四维筛选系统

| 维度 | 检查项 | 得分 |
|:---|:---|:---|
| **机构质量** | 顶级机构≥2家 | 0-3分 |
| **买入力度** | 金额>1亿，占比>15% | 0-3分 |
| **位置安全** | 历史低位，低估 | 0-3分 |
| **后续表现** | 历史胜率高 | 0-2分 |

**总分8分，6分以上进入观察池。**

#### 只盯三种机构票

| 类型 | 特征 | 频率 | 胜率 | 涨幅 |
|:---|:---|:---|:---|:---|
| **顶级机构重仓** | 社保+养老+QFII，持仓>5% | 低 | 80% | 30-50% |
| **主力机构扫货** | 龙虎榜多家机构，>1亿 | 中 | 70% | 15-25% |
| **北向连续加仓** | 连续5日，累计>2亿 | 高 | 65% | 10-20% |

#### 每日10分钟筛选流程

```
🎯 每日机构票扫描流程 🎯

【龙虎榜机构票】（3分钟）
- 只看机构专用席位
- 金额>5000万
- 多家机构同时出现

【北向连续加仓】（3分钟）
- 连续5日以上净买入
- 累计买入>2亿
- 股价未大涨

【大宗交易溢价】（2分钟）
- 溢价成交
- 机构买入
- 折价<5%

【筛选打分】（2分钟）
- 四维评分
- 6分以上入选
- 观察池不超过10只
```

|:---|:---|
| 看所有机构票 | 只盯最好的机构 |
| 盲目跟随 | 筛选后跟随 |

| 胜率不确定 | 胜率70%+ |

**不要看所有机构，只看最好的机构。不要跟所有机构票，只跟最强的信号。**
