## 三、持仓心理模块

### 3.1 五大心理弱点

| 心理问题 | 触发条件 | 系统建议 |
|----------|----------|----------|
| **恐高症** | 盈利>20% | 趋势未破坏，继续持有 |
| **贪婪症** | 盈利>30% | 执行分批止盈计划 |
| **犹豫症** | 接近止损位 | 严格执行止损纪律 |
| **侥幸症** | 亏损>2% | 立即止损，不抱幻想 |
| **后悔症** | 卖后继续涨 | 分析决策是否正确 |

### 3.2 心理分析流程

```
持仓数据 + 市场数据 → 心理状态分析 → 行动建议 → 坚定执行
```

## 四、主升浪强度评估

| 等级 | 涨幅预期 | 特征 | 持仓策略 |
|------|----------|------|----------|
| **S级** | 2-3倍 | 量能完美、斜率陡峭、板块共振、资金持续 | 重仓持有，不止盈 |
| **A级** | 50-100% | 多项指标配合 | 主要仓位持有 |
| **B级** | 20-30% | 指标一般 | 正常仓位 |
| **C级** | <10% | 指标较弱 | 轻仓或观望 |

## 五、动态止盈策略

| 策略 | 触发条件 | 操作 |
|------|----------|------|
| **移动止损** | 盈利>10% | 止损移至成本价 |
| **分批止盈** | 盈利>20% | 锁定一半利润 |
| **目标止盈** | 达到预测目标 | 分批减仓 |
| **信号止盈** | 强度明显下降 | 立即止盈 |
