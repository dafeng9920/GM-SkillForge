## 二、Week 5: 技能资产化系统

### 2.1 核心能力

| 能力 | 说明 |
|------|------|
| **从复盘中学习** | 每笔交易生成作业，次日验证 |
| **从作业中学习** | 批改作业，分析错题 |
| **自动更新进化** | 根据错题规律调整参数 |
| **知识可查询** | 所有经验可查询可复用 |
| **技能可移植** | 一个股票学到，其他股票也能用 |

### 2.2 学习闭环

```
交易决策 → 生成预测作业 → T+1验证 → 错题分析 → 参数迭代 → 下次更好
```

### 2.3 知识库结构

```python
class SkillKnowledgeBase:
    """技能知识库"""

    def __init__(self):
        self.skills = {
            '突破识别': {...},
            '洗盘识别': {...},
            '诱多识别': {...},
            '主升浪判断': {...},
            # ...
        }
        self.performance = {}  # 每个技能的准确率
        self.updates = []      # 历史更新记录
```

### 2.4 知识进化机制

| 类型 | 触发条件 | 动作 |
|------|----------|------|
| **微调** | 准确率>80% | 参数微调±10% |
| **优化** | 准确率50-80% | 重新优化参数 |
| **重建** | 准确率<50% | 重新训练模型 |
