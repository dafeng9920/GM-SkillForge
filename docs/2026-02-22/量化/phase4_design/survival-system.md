## 二、四大生存机制

### 2.1 生存线保障

| 级别 | 资金比例 | 状态 | 操作 |
|------|----------|------|------|
| 安全区 | 100%-70% | 安全 | 正常交易 |
| 警告线 | 70% | 警告 | 仓位降至20%，暂停开新仓 |
| 危险线 | 50% | 危险 | 仓位降至10%，只做最确定信号 |
| 生死线 | 30% | 生死 | 立即平仓，转出资金，全面复盘 |
| 出局线 | 10% | 出局 | 强制停止，永远离场 |

### 2.2 熔断机制

| 类型 | 触发条件 | 操作 |
|------|----------|------|
| 连续亏损熔断 | 连续3次亏损 | 暂停交易，复盘 |
| 日熔断 | 单日亏损>5% | 当日停止 |
| 周熔断 | 单周亏损>10% | 本周停止 |
| 月熔断 | 单月亏损>15% | 本月停止 |

### 2.3 黑天鹅防护

| 情况 | 特征 | 保护模式 |
|------|------|----------|
| 闪电崩盘 | 5分钟跌>3% | 清仓模式 |
| 千股跌停 | 跌停家数>10% | 清仓模式 |
| 大盘熔断 | 单日跌>5% | 清仓模式 |
| 流动性危机 | 成交量萎缩>50% | 防守模式 |

### 2.4 紧急模式

| 生存分数 | 状态 | 操作 |
|----------|------|------|
| 70-100 | 安全 | 正常交易 |
| 50-70 | 注意 | 谨慎交易 |
| 30-50 | 警告 | 降低仓位 |
| <30 | 紧急 | 立即停止 |
