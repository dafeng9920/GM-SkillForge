# 策略自生成系统

> **核心理念**: 系统不仅能执行策略，还能根据市场状态**创造和进化策略**。

## 一、从执行者到创造者

| 普通系统 | 策略自生成系统 |
|:---|:---|
| 固定策略，应对所有市场 | 根据市场状态，动态生成策略 |
| 策略老化，需要人工更新 | 策略自动进化，永不过时 |
| 回测历史，优化参数 | 理解市场，创造新策略 |
| 依赖人的经验 | 超越人的经验 |

## 二、系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    策略自生成系统                              │
│                                                             │
│  市场基因解析器 → 策略工厂 → 策略验证器 → 策略进化器        │
│        ↓                                                        │
│  策略知识库 (历史策略、市场映射、绩效记录)                   │
└─────────────────────────────────────────────────────────────┘
```

## 三、市场基因解析

| 基因 | 分析内容 | 产出 |
|------|----------|------|
| **趋势基因** | 方向、强度、持续性 | up/down, strength |
| **波动基因** | 水平、结构、聚集 | low/normal/high/extreme |
| **流动性基因** | 成交量变化 | very_low/low/normal/high/very_high |
| **情绪基因** | 恐惧贪婪 | fear/neutral/greed |
| **结构基因** | 支撑阻力、形态 | trend/range/breakout |
| **周期基因** | 周期阶段 | early/mid/late |

## 四、策略模板库

| 模板 | 适用市场 | 核心逻辑 |
|------|----------|----------|
| **趋势策略** | steady_bull/normal_bull | 跟随趋势，顺势而为 |
| **均值回归** | quiet_range/chaotic_range | 超买超卖，回归均值 |
| **突破策略** | tight_range | 突破确认，顺势入场 |
| **动量策略** | volatile_bull/volatile_bear | 追涨杀跌，动量延续 |
| **波动策略** | high volatility | 利用波动，赚取波动率 |
| **套利策略** | 特定条件 | 统计套利，市场中性 |

## 五、进化机制

| 绩效水平 | 进化方式 | 操作 |
|----------|----------|------|
| 优秀 (Sharpe > 1.5) | 微调 | 参数变异 ±20% |
| 一般 (Sharpe 0.5-1.5) | 多种进化 | 尝试参数变异、组件交换、添加/移除组件 |
| 较差 (Sharpe < 0.5) | 大改 | 重新生成策略 |
